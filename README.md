Description:
Phantom Clicker is a high-performance automation tool 
designed for precision and speed. This version is fully 
optimized for low CPU usage and maximum reliability.

Instructions:
1. Run "Phantom_Clicker.exe".
2. Set your desired click interval (in milliseconds).
3. Use the configured Hotkeys to Start and Stop the clicker.
4. Make sure to "Run as Administrator" if the game or 
   application you are using requires it.

Security Note (Antivirus):
Since this tool is built with Python and compiled into a 
standalone executable (.exe), some antivirus software may 
flag it as a "False Positive". Rest assured, Phantom Clicker 
is 100% safe and contains no malicious code.
